ORGANIZA ELIZA — IMAGES FOLDER

Drop your photos in this folder using these exact filenames and they'll
appear automatically on the site. Until then, you'll see soft placeholder
tiles where each image will go.

TIPS
  - JPG or PNG both work. Keep each file under ~500 KB so the site loads fast
    (you can shrink photos free at squoosh.app).
  - Want more or fewer gallery photos? Add/remove the matching blocks in
    gallery.html — each one is clearly labeled.
